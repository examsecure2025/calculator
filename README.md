# Calculator

## Description

A simple and functional calculator application that performs basic mathematical operations including addition, subtraction, multiplication, and division. The calculator displays expressions at the top and shows results in the main display area. It features a clean, modern interface with responsive design that works on all screen sizes.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Expression display showing current operation
- Clear button to reset calculations
- Loading animations to prevent double clicks
- Responsive design for mobile and desktop
- Error handling for invalid operations

## How to Run

1. Open the 'calculator' folder
2. Double-click on 'index.html' file
3. The calculator will open in your default web browser
4. You can start using the calculator immediately

## Files

- index.html - Main HTML structure
- style.css - Styling and layout
- script.js - Calculator functionality
